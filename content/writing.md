## How I publish

1. Create a new file in `content/writing/` with front matter like:
   ```markdown
   ---
   title: My thoughtful essay
   description: A short teaser to show up in previews.
   tags: [writing]
   ---
   ```
2. Write in Markdown, embed images or code snippets like you would in Obsidian.
3. Commit and deploy — the list pages, graph, and search update automatically.
